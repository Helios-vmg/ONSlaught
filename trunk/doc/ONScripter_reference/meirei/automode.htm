automode(mode_ext)

This is a command developed especially for the commercial software 「Gin'iro」. When you use this command, you can enter Auto Mode from the menu. It is similar in concept to the 「skip to next choice」 option, but it displays characters much more slowly (with a delay defined by automode_time). So it is almost like a mode in which the computer reads the text to you.

You can effectively adjust sound playback volume from the menus while Auto Mode is still progressing (MP3(BGM) voice(DWAVE 0) SE(DWAVE 1〜) are all still active, and adjustable).

There are future plans to add further functionality to this command, and to make things more rigorously conformant to predefined syntaxes.
