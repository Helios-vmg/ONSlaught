prnum

prnum <numerical value label number>,<current value>,x,y,<character width>,<character height>,<color>

　Creates a numerical value label, and then blits its current value in the given position at the given size and color.
　Its value ranges 3 columns, so from 0-999.
　You need to call a print command or something like it to actually display this onscreen.


bar and prnum were specifically created for mode_saya, which was inturn specifically created for the doujin game 「Saya 〜Labyrinth of Immorality〜」.
There are currently plans to replace this series of commands with something a little bit more generalized and powerful.
