setcursor

setcursor <character string>(<character string>),<number>,<number>

Loads and displays a cursor definition file for use during click wait state.
The character strings are, from left to right, cursor for normal wait state and cursor for page feed wait state.
The numbers comprise an (x,y) coordinate that is positioned relative to the upper left pixel of the first choice text.
By default, these variables are:
